# Welcome to "mui-material-components-tsdx"

The MuiMaterialTSDX Components is a set of React components based on Material Design.

Browse example stories now by navigating to them in the sidebar.
You can check the available components in the sidebar.

## Get started

To install the MuiMaterialTSDX Components in your project, run:

```bash
npm install mui-material-components-tsdx
```

or

```bash
yarn add mui-material-components-tsdx
```

## How to use

To use the Material Components in your project, add the following import:

```bash
    import { FitButton } from 'mui-material-components-tsdx';
```

And then you can use the component:

```jsx
<FitButton>Click me!</FitButton>
```

## Testes

To run the tests, run:

```bash
npm run test
```

or

```bash
yarn test
```
